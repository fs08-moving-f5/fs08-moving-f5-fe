## 🎯 서비스 소개

**무빙(Moving)**은 이사 전문가와 소비자를 효율적으로 연결하는 매칭 플랫폼입니다.

> 프로젝트 기간: 2025. 12. 05. - 2026. 01. 26.

### 주요 기능

- **견적 요청 및 비교**: 원하는 이사 조건을 입력하고 여러 기사님의 견적을 한눈에 비교
- **전문가 매칭**: 지역, 서비스 유형, 경력 등을 기반으로 최적의 이사 전문가 추천
- **후기 및 평점**: 실제 고객 후기와 평점을 통해 신뢰할 수 있는 전문가 선택
- **즐겨찾기**: 마음에 드는 기사님을 즐겨찾기에 추가하여 나중에 쉽게 찾기
- **실시간 알림**: 견적 요청, 수락, 거절 등 실시간 알림으로 빠른 소통
- **프로필 관리**: 사용자와 기사님 모두 자신의 프로필을 관리하고 업데이트

### 서비스 유형

- 일반 이사 (원룸, 투룸, 아파트 등)
- 기업/사무실 이사
- 포장 이사

### 주요 디렉토리 설명

- **`src/app/`**: Next.js App Router 기반 라우팅 및 레이아웃
- **`src/features/`**: 기능별로 분리된 모듈 (각 모듈은 `hooks/`, `services/`, `types/`, `ui/` 포함)
- **`src/shared/`**: 프로젝트 전반에서 공유되는 유틸리티, 컴포넌트, 타입
- **`src/views/`**: 페이지 레벨 컴포넌트

## 🚀 시작하기

### 사전 요구사항

- Node.js 18.x 이상
- npm 또는 yarn

### 설치

```bash
# 의존성 설치
npm install
```

### 환경 변수 설정

`.env.example` 파일을 참고하여 `.env` 파일을 생성하고 필요한 환경 변수를 설정하세요.

```bash
cp .env.example .env
```

### 빌드

```bash
# 프로덕션 빌드
npm run build

# 프로덕션 서버 시작
npm start
```

### 기타 스크립트

```bash
# 린트 검사
npm run lint

# 타입 체크
npm run type-check

# 코드 포맷팅
npm run format

# OpenAPI 타입 생성
npm run generate:types
```
